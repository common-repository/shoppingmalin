== Description ==

Cette extension vous permet d'installer le comparateur de prix ShoppingMalin dans votre site.

Vous pouvez installer 3 formes diff&eacute;rentes :<br>
- Comparateur ShoppingMalin, l'internaute va saisir une r&eacute;f&eacute;rence et lancer une recherche<br>
- R&eacute;f&eacute;rence sp&eacute;cifique, vous choisissez une r&eacute;f&eacute;rence, le plugin affiche les prix des diff&eacute;rents marchands qui proposent cet article<br>
- Meilleur prix, vous choisissez une r&eacute;f&eacute;rence, le plugin affiche le nom du marchand le moins cher et son prix.<br>

Il est possible de s'inscrire chez Amazon pour gagner une commission sur les ventes.
Vous serez r&eacute;mun&eacute;r&eacute; si une vente est r&eacute;alis&eacute;e apr&egrave;s le clic d'un d'internaute sur votre site.

== Installation ==

1. Uploadez les fichiers dans le r&eacute;pertoire `/wp-content/plugins/shoppingmalin`, ou installez le plugin directement depuis la page Extensions du site Wordpress.
2. Activez le plugin depuis votre administration Worpress, menu Exensions.
3. Utilisez le menu "R&eacute;glages" -> "ShoppingMalin" pour configurer le plugin
4. Collez le shortcode dans un de vos articles ou dans une de vos pages
